CryptoBot

  • Personnel
  • Node.js
  • Heroku
  • API

CryptoBot est un BOT Discord. Un BOT Discord va "écouter" les messages d'un canal de texte Discord et réagir en conséquence de certains mots clés.
La fonctionnalité primaire de CryptoBot est d'afficher le prix d'une cryptomonnaie demandée.

CryptoBot est né d'une envie d'apprendre plusieurs technologies et de proposer une solution rapide d'accéder à la donnée en temps réel.

Les objectifs de ce projet ont été :

  • D'apprendre le Node.js
  • Apprendre à déployer une application
  • Utiliser des API (Discord, Coingecko)

Code source : https://github.com/HongThomas/CryptoBot

Secret Santa

  • Personnel
  • Python
  • smtplib

Secret santa est un script Python qui prend une liste d'adresses mails en entrée et effectue un envoi d'emails aléatoire et non répétitif.

Ce projet est né d'un besoin avec mes amis d'utiliser une solution anonyme de s'attribuer un receveur de cadeau pour Noël.

Les objectifs de Secret Santa :

  • Dans le contexte du covid, trouver une solution intermédiaire
  • Appliquer les notions d'algorithmique apprises en S1 du DUT informatique
